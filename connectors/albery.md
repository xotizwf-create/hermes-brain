# Connector: Албери (MCP)

MCP-сервер `mcp.m4s.ru`, подключён как `albery` (HTTP, auth none, секрет в URL).
Человеческое имя для владельца — **«Албери»**; в командах включения/выключения/удаления это
маппится на id `albery`.

## Назначение

Контекстный сервер Албери: Bitrix-задачи и чаты, Zoom-транскрипты, база знаний компании,
инструкции для ИИ, отчёты владельца и операционные рекомендации.

## Инструменты

На момент подключения сервер отдаёт 51 инструмент (`mcp_albery_*`). Общие группы:
- Старт/инструкции: обязательная загрузка AI-инструкций, статус, контекстные гайды.
- Знания компании: профиль, файлы, поиск по базе знаний, структура, периоды и готовность отчётов.
- Bitrix: поиск задач, комментарии, создание/удаление задач, чаты и сообщения.
- Zoom: список звонков, транскрипты, экспорт, поиск, отчёты.
- Отчёты и рекомендации владельца: дневные/недельные отчёты, рекомендации, рассылки в Bitrix.
- Служебные действия: чтение URL, обновление AI-инструкций, компактный экспорт.

## Без подтверждения

- Чтение/поиск: инструкции, статусы, контекст, файлы, база знаний, задачи, чаты, Zoom-транскрипты,
  отчёты и рекомендации.

## Только с подтверждением

- Внешние действия и изменения: создание/удаление Bitrix-задач, отправка сообщений, рассылка отчётов,
  сохранение/удаление отчётов, отправка рекомендаций, обновление AI-инструкций.

## Подключение и управление

Через скилл `connect-mcp` (`hermes_mcp.py`). Секрет хранится только в live-конфиге Hermes (600);
в этом репозитории секрета нет, только ссылка по имени. Реестр: `connectors/registry.yaml`.
